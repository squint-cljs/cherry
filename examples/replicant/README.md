# replicant tic-tac-toe

[Replicant](https://github.com/cjohansen/replicant) tic-tac-toe compiled by
cherry, served by vite with the cherry plugin (browser REPL included).

Run:

    npm install
    npm run dev

Build:

    npm run build

Note: depends on borkdude's replicant fork, which treats `:cherry` like
`:squint` in the `with-dev-key` reader conditional (replicant.dom); pending
upstream.
